    Installation Guide

    Prerequisites

    Before installing AlmaSEO, ensure your WordPress site meets these requirements:

    • WordPress Version: 5.0 or higher
    • PHP Version: 7.4 or higher
    • Memory Limit: 256MB minimum (512MB recommended)
    • Active Theme: Any WordPress theme
    • Plugin Compatibility: Compatible with most SEO plugins

    Installation Methods

    Method 1: WordPress Admin (Recommended)

    1. Log into WordPress Admin
    2. Navigate to your WordPress dashboard

    3. Go to Plugins → Add New

    4. Search for AlmaSEO

    5. In the search box, type "AlmaSEO"
    6. Look for "AlmaSEO - AI Content Generator"

    7. Click Install Now

    8. Activate the Plugin

    9. After installation, click Activate
    10. You'll see a new "AlmaSEO" menu in your admin sidebar

    Method 2: Manual Upload

    1. Download the Plugin
    2. Download the AlmaSEO plugin ZIP file

    3. Extract the contents to a folder

    4. Upload via FTP

    5. Connect to your server via FTP
    6. Navigate to /wp-content/plugins/

    7. Upload the almaseo folder

    8. Activate in WordPress

    9. Go to Plugins in your WordPress admin
    10. Find "AlmaSEO" and click Activate

    Initial Setup

    1. API Key Configuration

    After activation, you'll need to configure your API key:

    1. Navigate to AlmaSEO Settings
    2. Go to AlmaSEO → Settings

    3. Click on the API Configuration tab

    4. Enter Your API Key

    5. Paste your AlmaSEO API key
    6. Click Save Settings

    2. Site Verification

    1. Test Connection
    2. Click Test Connection to verify your setup

    3. Ensure all systems are communicating properly

    4. Verify Permissions

    5. Check that AlmaSEO has proper permissions
    6. Ensure your hosting environment supports the required features

    Troubleshooting

    Common Issues

    Plugin Not Activating - Check PHP version compatibility - Verify WordPress version requirements - Ensure sufficient memory allocation

    API Connection Errors - Verify your API key is correct - Check your internet connection - Ensure your hosting allows external API calls

    Content Generation Fails - Check your API quota - Verify your content settings - Review error logs for specific issues
