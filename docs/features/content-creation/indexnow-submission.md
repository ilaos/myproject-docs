# IndexNow Submission

**Tier:** Pro | **Type:** Integration

!!! tip "Key Insight"
    Paired with Search Console Indexing, AlmaSEO covers all major search engines. Most users don't even know IndexNow exists — this is "SEO expertise baked in." The setup guide in the UI reduces friction. Marketable as: "Your content is submitted to Google, Bing, and Yandex the moment it's published.”

## Overview

Instantly notifies participating search engines (Bing, Yandex, and others) when content is published or updated. User sets up by generating a key at http://indexnow.org/, creating a verification file, uploading to site root, and saving the API key in AlmaSEO. Once configured, one-click submission pings all IndexNow participants.

## Why It Matters

Instantly notifies Bing, Yandex, etc. when content is published.

## Requirements

- **Integrations:** IndexNow
- **Compatibility:** Wordpress
