# Writing Tone

**Tier:** Core/Free | **Type:** User-Facing UI

!!! tip "Key Insight"
    Simple but expected. Users want control over voice. Consider future enhancement: save default tone to Client Profile so it's pre-selected but still adjustable per article.

## Overview

Single-select tone setting for each article. Options: Professional, Friendly, Authoritative, Conversational. Affects word choice and voice but not structure. Selected per article, not saved to profile.

## Why It Matters

Single-select tone setting. Affects word choice, not structure.

## Requirements

- **Integrations:** OpenAI API

## Target Audience

Agencies, DIY Business Owners, Freelancers
