# Wordpress Categories

**Tier:** Core/Free | **Type:** Integration

!!! tip "Key Insight"
    Small but meaningful time-saver. Keeps the user inside AlmaSEO instead of bouncing to WordPress to fix tags. Also feeds Underserved Category Detection — the same category list powers the "you have fewer than 3 posts in this category" insight.

## Overview

 Pulls existing categories from the user's WordPress site via API and displays them as selectable options. User chooses which categories apply to the article. On publish, categories are automatically assigned to the post — no manual tagging required in WordPress.

## Why It Matters

Pulls existing WordPress categories via API.

## Requirements

- **Integrations:** Wordpress API
- **Compatibility:** Wordpress

## Target Audience

Agencies, Freelancers
