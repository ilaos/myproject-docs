# Static Site Export Panel

**Tier:** Core/Free | **Type:** User-Facing UI

!!! tip "Key Insight"
    Static Site Export is the non-WordPress differentiator. The UI makes it dead simple — pick your format, download or copy, grab the image, done. Supports Hugo, Astro, Next.js, Jekyll, 11ty, and MDX out of the box. Quick Copy mode bridges the gap for Squarespace/Wix users. Multi-location businesses get smart content — single location pulls full NAP data, multiple locations keep articles clean with just city names hyperlinked to each Google Maps business page. Should be prominently featured when targeting non-WordPress audiences.

## Overview

The export interface shown to static site users after content is generated. Displays all Multi-Format Export options (Markdown, HTML Full Page, HTML Body, JSON, MDX, Structured JSON, Structured YAML) with Download and Copy to Clipboard buttons. Includes Featured Image download with placeholder replacement instructions (<!-- ALMASEO_IMAGE: ... -->), Quick Copy buttons (HTML Content, Markdown, SEO Metadata, URL Slug), and expandable SEO Metadata Preview. Replaces WordPress-specific features (IndexNow, Search Console Indexing) which require a live published URL.

## Why It Matters

Complete export solution for static site users (Markdown, JSON, etc).

## Requirements

- **Integrations:** OpenAI API
- **Compatibility:** Static Sites

## Target Audience

DIY Business Owners, Freelancers
