**Tier:** Pro | **Type:** Integration

## Overview

One-click submission to Google Search Console for faster discovery. "Send to Search Console" button submits your sitemap, pings Google, and provides URL inspection link. Info note explains the 24-72 hour indexing window based on content quality and Google's discretion.
