# Structured Data (Schema)

**Tier:** Pro | **Type:** Automation / Background

!!! tip "Key Insight"
    Schema is implemented in multiple places for different workflows (copy/paste tools, exporters, injection pipelines, editor plugins, and detection/scoring). Treating it as a single capability in the map prevents schema improvements from getting lost inside unrelated features.

## Overview

Umbrella capability for generating, validating, detecting, and delivering http://Schema.org structured data across AlmaSEO. This covers multiple schema systems including on-demand generators, export-time embedding, WordPress injection, editor integrations, and background detection/scoring.

## Why It Matters

Improves eligibility for rich results and reinforces trust signals by ensuring the site’s structured data is present, correct, and aligned with real business information.

## Target Audience

Agencies, DIY Business Owners, Freelancers

## Sub-features

- **[Structured Data Dashboard & WordPress Injection](structured-data-dashboard-wordpress-injection.md)** *(Agency)* — API-driven structured data system with optional dashboard UI (behind SD_ENABLE=1) that can generate schema snippets, track coverage, store results,...
