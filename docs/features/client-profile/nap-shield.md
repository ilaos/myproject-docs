# NAP Shield

**Tier:** Agency | **Type:** User-Facing UI

!!! tip "Key Insight"
    Verification-first citation audit: you set citation statuses (verified, needs_update, fixed, dismissed, not_checked) and the health score is computed from your confirmations. Supports multi-location, SABs (name+phone), acceptable address variants, runtime flags, and feeds SEO Recovery 911 as an audit signal.

## Overview

Monitors a business’s Name, Address, Phone (NAP) consistency across the web by discovering citations, letting you review and track fixes, and computing a verification-based health score.

## Why It Matters

Helps improve local SEO trust and rankings by finding and fixing inconsistent NAP listings, with clear progress tracking per location.

## Requirements

- **Integrations:** OpenAI API

## Access Control

**Permission Mode:** Configurable per client

## Target Audience

Agencies, DIY Business Owners, Freelancers
