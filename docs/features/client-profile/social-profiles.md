# Social Profiles

**Tier:** Pro | **Type:** User-Facing UI

!!! tip "Key Insight"
    Writers can automatically pull official social profile URLs into copy snippets (e.g., “Connect on Facebook/Instagram/LinkedIn”) from the Client Profile source of truth.

## Overview

A centralized place to store and manage a client’s social media profile links so they’re easy to reference and reuse (including by the article writer).

## Why It Matters

Keeps all client social links in one spot and makes it easy to include accurate profile links in written content (e.g., “Follow us on Facebook”).

## Target Audience

Agencies, DIY Business Owners, Freelancers
