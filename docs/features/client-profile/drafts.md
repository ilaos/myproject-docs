# Drafts

**Tier:** Pro | **Type:** User-Facing UI

!!! tip "Key Insight"
    Drafts is optimized for momentum: a clean grid of works-in-progress plus a single “continue editing” entrypoint reduces friction and keeps the team in the same post detail workflow used for publishing, exports, and QA.

## Overview

UI elements:
- Total draft count badge
- Write New Article button
- 3-column card grid (up to 9 drafts) showing:
	- Draft status badge
	- Truncated title (50 chars)
	- Meta description excerpt (80 chars)
	- Last modified date
	- Continue Editing button linking to the post detail page

## Why It Matters

Keeps in-progress content organized and easy to resume, with quick access back into the post detail editing workflow.

## Requirements

- **Integrations:** OpenAI API, Wordpress API
