# AlmaSEO for WordPress

## Overview

A Client Profile tab that shows the status and health of the AlmaSEO plugin installed on a connected WordPress site.

## What It Does

This tab checks which AlmaSEO plugin — whether the Connector Plugin or the SEO Playground — is active on the connected WordPress site and reports on its status. It provides a quick health check so users can verify the plugin is installed, active, and functioning correctly without needing to log into WordPress.

Key information displayed:

- Which AlmaSEO plugin is detected (Connector or Playground)
- Plugin version
- Connection status (active, stale, or disconnected)
- Last successful communication timestamp

## Why It Matters

- **At-a-glance health check** — instantly see if the AlmaSEO plugin on a client's WordPress site is working properly.
- **No WordPress login required** — verify plugin status directly from the AlmaSEO dashboard.
- **Proactive troubleshooting** — catch stale connections or outdated plugin versions before they cause publishing failures.

## How to Use It

1. Open a site's **Client Profile**.
2. Navigate to the **AlmaSEO for WordPress** tab.
3. Review the detected plugin type, version, and connection status.
4. If the status shows issues, follow the prompts to reconnect or update the plugin.

## Notes / Edge Cases

- Requires either the AlmaSEO Connector Plugin or the AlmaSEO SEO Playground to be installed and activated on the WordPress site.
- If neither plugin is detected, the tab will indicate that no AlmaSEO plugin is found and prompt for installation.
